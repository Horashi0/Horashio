Rules for using the streaming section

- Never move layouts when control panel is open


- If stream layouts move when you open control panel dont worry


- Load layouts in the way you saved. e.g. if you saved a layout with nav bar up and fullscreen, load it with the nav bar up and fullscreen


- If the stream layouts aren't saving, check your cookie settings. It might be that your browser is deleting them
